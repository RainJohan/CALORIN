🎮 Game Title:

Veil of Curses — Jujutsu Kaisen Tower Defense

📜 Game Description:

Veil of Curses is a tower defense game that is inspired by anime and is set in the world of Jujutsu sorcery. Players are in control of strong sorcerers and tactically place them on a grid to keep the waves of deadly curses off.

Collect cursed energy (CE), call on iconic characters such as Yuji, Gojo, and Sukuna, and withstand more challenging waves of enemies. The sorcerers possess different powers, each possessing certain abilities; some of them create energy, some attack, others defend or cause calamities of their domains.

Defend the wall with plan, time and resource allocation to keep the curses at bay before they overpower your defenses.

🎮 List of Controls

🖱 Mouse Controls
Left Click (on shop) → Select a sorcerer
Left Click (on grid) → Place selected sorcerer
Left Click (on ✦ CE orbs) → Collect Cursed Energy
Left Click (with Remove selected) → Remove a placed sorcerer

🎯 Buttons
⏸ Pause Button → Pause / Resume the game
↺ Restart Button → Restart the game from the beginning

🧠 Gameplay Actions
Select Sorcerer First → Then click a tile to deploy
Manage CE (Cursed Energy) → Needed to place units
Defend the Barrier → Don’t let curses pass the left side
Survive Waves → Endless mode

⚠️ Special Controls
Sukuna (💀) → Automatically explodes after a few seconds (no manual trigger)
Gojo (♾) → Attacks automatically and slows enemies
Shoko (🌸) → Generates CE over time (no clicking needed)
